# Aryxzing Portfolio

A premium one-page portfolio website for Aryxzing, a freelance web designer and full-stack developer, built with React, Vite, Tailwind CSS, Framer Motion, and Lucide React.

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal, usually:

```bash
http://localhost:5173
```

## Production Build

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```

## Project Structure

```text
src/
  components/
    sections/
    ui/
  data/
  App.jsx
  index.css
  main.jsx
```

## Notes

- All portfolio content is sample content and can be edited in `src/data/content.js`.
- The project is designed as a polished single-page experience with responsive layouts, animated reveals, a project modal, and premium background effects.
